# Loan-approval-project
# 🏦 Loan Approval Data Analysis Project

## 📌 Project Overview
This project focuses on analyzing loan approval data to identify key factors that influence loan approval decisions. The analysis includes data cleaning, exploratory data analysis (EDA), and insights generation using Python.

---

## 📊 Dataset Information
- Total Records: 367
- Features:
  - ApplicantIncome
  - CoapplicantIncome
  - LoanAmount
  - Loan_Amount_Term
  - Credit_History
  - Gender, Married, Education, Property_Area, etc.

---

## 🛠️ Tools & Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 🔍 Project Workflow

### 1. Data Cleaning
- Handled missing values using:
  - Mode (categorical data)
  - Median (numerical data)
- Removed all null values successfully

### 2. Exploratory Data Analysis (EDA)
- Univariate Analysis (Histograms, Boxplots)
- Bivariate Analysis (Scatter plots, Pair plots)
- Multivariate Analysis (Correlation heatmap)

---

## 📈 Key Insights

- 📌 Applicant Income has a **moderate impact** on Loan Amount
- 📌 Co-applicant income has **very low influence**
- 📌 Most applicants have income between **₹5,000 – ₹10,000**
- 📌 Male applicants dominate (~81%)
- 📌 Loan amount generally increases with income (not always)

---

## 📊 Visualizations
- Histograms
- Boxplots
- Bar Charts
- Pie Charts
- Heatmap

---

## 🧠 Conclusion
Applicant income is the most important factor affecting loan amount. Other features like co-applicant income and loan term have minimal impact. The dataset shows a strong imbalance in gender distribution and income patterns.
